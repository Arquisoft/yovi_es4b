:toc:
:toc-title: Indice

Esta documentacion esta escrita como guia de estudio personal para preparar la presentacion del proyecto.

La idea no es describir cada funcion, sino darte una comprension global y profunda del sistema:

- como se divide
- como se comunican las piezas
- donde esta la logica realmente importante
- como funciona el online
- que decisiones de arquitectura hay detras
- que limites tiene la implementacion actual
